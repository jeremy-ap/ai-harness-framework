## How to Add a Runbook

1. Copy [TEMPLATE.md](./TEMPLATE.md) to a new file: `NNN-short-title.md`
2. Fill in all sections — every step must be explicit and executable
3. Add an entry to the table above
4. Test the runbook by following it step by step
5. Record the date in "Last Verified"

## Guidelines

- Runbooks must be executable by both humans and agents
- Every step must include the exact command or action to take
- Include verification steps after critical actions
- Include rollback procedures for destructive operations
- Review and re-verify runbooks quarterly
