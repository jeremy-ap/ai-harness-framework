## How to Add an Item

1. Copy [TEMPLATE.md](./TEMPLATE.md) to a new file: `NNN-short-title.md`
2. Fill in all sections
3. Add an entry to the table above
4. Prioritize during the next planning session

## Severity Levels

- **Critical** — Actively causing bugs or outages. Fix immediately.
- **High** — Significantly slows development or risks reliability. Fix soon.
- **Medium** — Noticeable friction but workarounds exist. Plan to fix.
- **Low** — Minor annoyance. Fix when convenient.

## Statuses

- **Open** — Identified but not yet being addressed
- **In Progress** — Actively being worked on
- **Resolved** — Fixed and verified
- **Won't Fix** — Accepted as-is (document the rationale)
