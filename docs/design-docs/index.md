## How to Add a Design Doc

1. Copy [TEMPLATE.md](./TEMPLATE.md) to a new file: `NNN-short-title.md`
2. Fill in all sections
3. Add an entry to the table above
4. Submit for review

## Statuses

- **Draft** — Work in progress, not yet proposed
- **Proposed** — Ready for review and discussion
- **Accepted** — Approved and in effect
- **Rejected** — Reviewed and declined (kept for historical record)
- **Superseded** — Replaced by a newer design doc (link to replacement)
