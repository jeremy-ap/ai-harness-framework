# Generated Documentation

This directory holds auto-generated documentation files. **Do not manually
edit files in this directory** — they will be overwritten the next time a
generator script runs.

## What Generates Into Here

| File | Generated By | Description |
|------|-------------|-------------|
| `quality-score.md` | `scripts/quality-score.sh` | Overall project quality metrics |
| `dependency-graph.md` | `scripts/generate-dependency-graph.sh` | Module dependency visualization |

## How to Regenerate

Run the relevant script from the project root:

```bash
./scripts/quality-score.sh
./scripts/generate-dependency-graph.sh
```

Or regenerate everything:

```bash
./scripts/quality-score.sh && ./scripts/generate-dependency-graph.sh
```

## Adding a New Generator

If you create a new script that generates documentation:

1. Have the script write output to `docs/generated/`
2. Add the file and script to the table above
3. Add the generated file to `.gitignore` if it should not be committed
   (or commit it if it should be tracked for agent consumption)
